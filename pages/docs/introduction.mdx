## 🛠️ **Technology Stack**

Our documentation portal is built with modern, cloud-native technologies:

### **Frontend & Documentation**
- **Next.js 15** - React framework with App Router
- **Nextra** - Documentation theme for Next.js
- **MDX** - Markdown with JSX components
- **TypeScript** - Type-safe development
- **TailwindCSS** - Utility-first CSS framework

### **Authentication & Security**
- **NextAuth.js** - Authentication framework
- **GitHub OAuth** - Organization-based access control
- **Protected Routes** - Secure content access

### **Infrastructure & Deployment**
- **Docker** - Containerization with multi-stage builds
- **Kubernetes** - Container orchestration
- **OpenShift** - Enterprise Kubernetes platform
- **Helm** - Package management for Kubernetes
- **GitHub Actions** - CI/CD automation

### **Monitoring & Observability**
- **Prometheus** - Metrics collection and storage
- **Grafana** - Metrics visualization and dashboards
- **OpenShift Monitoring** - Built-in monitoring stack
