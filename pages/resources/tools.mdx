# Tools & Utilities

Discover essential tools and utilities for managing Cloud Native environments, monitoring performance, and troubleshooting issues. Our curated collection includes both official tools and community-contributed utilities.

## Command Line Tools

### OpenShift CLI (oc)
- **Installation**: Download from Red Hat Customer Portal
- **Key Commands**:
  ```bash
  oc login                    # Authenticate to cluster
  oc get pods                # List pods
  oc describe pod <name>     # Detailed pod information
  oc logs <pod-name>         # View pod logs
  oc exec -it <pod> -- bash # Execute commands in pod
  oc port-forward <svc> 8080:80 # Port forwarding
  ```

### Kubernetes CLI (kubectl)
- **Installation**: Multiple package managers available
- **Essential Commands**:
  ```bash
  kubectl get all            # List all resources
  kubectl apply -f file.yaml # Apply configuration
  kubectl delete -f file.yaml # Delete resources
  kubectl scale deployment <name> --replicas=3 # Scale deployment
  kubectl rollout status deployment/<name> # Check rollout status
  ```

### Helm Package Manager
- **Installation**: `curl https://get.helm.sh/helm-v3.x.x-linux-amd64.tar.gz | tar xz`
- **Common Operations**:
  ```bash
  helm repo add bitnami https://charts.bitnami.com/bitnami
  helm install my-app bitnami/nginx
  helm upgrade my-app bitnami/nginx
  helm uninstall my-app
  helm list
  ```

## Monitoring & Observability Tools

### Prometheus
- **Purpose**: Metrics collection and storage
- **Key Features**:
  - Time-series database
  - PromQL query language
  - Alert manager integration
  - Service discovery
- **Installation**: Operator-based deployment recommended

### Grafana
- **Purpose**: Metrics visualization and dashboards
- **Capabilities**:
  - Custom dashboard creation
  - Multiple data source support
  - Alerting and notifications
  - Team collaboration features
- **Access**: Web-based interface

### OpenShift Monitoring Stack
- **Built-in Components**:
  - Prometheus
  - Grafana
  - Alert Manager
  - Node Exporter
- **Configuration**: Via OpenShift Console or CLI
